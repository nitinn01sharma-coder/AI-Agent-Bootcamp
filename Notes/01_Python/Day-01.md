# Interview Questions

## Q1 What is Python?

Python is a high-level, interpreted, object-oriented programming language used for AI, ML, Web Development, Automation, and Data Science.

---

## Q2 What is pip?

pip is Python's package manager used to install, update, and remove packages.

Example:

```bash
pip install openai
```

---

## Q3 What is Virtual Environment?

A virtual environment is an isolated Python environment where project-specific dependencies are installed.

Benefits:
- Dependency isolation
- Version management
- No package conflicts

---

## Q4 Why do we use Git?

Git is a distributed version control system used to track code changes and collaborate with teams.

---

## Q5 Difference between Git and GitHub?

Git → Local Version Control

GitHub → Cloud Repository
